# 🐾 Vetty App

A full-stack veterinary e-commerce platform that allows users to purchase pet products and book veterinary services.

---

## 🚀 Project Overview

Vetty is designed to:
- Provide veterinary services (grooming, vaccination, checkups)
- Sell pet-related products (food, medicine, toys)
- Allow users to book services and track orders
- Provide an admin dashboard for managing the system

---

## 🛠️ Tech Stack

- **Frontend:** React + Redux Toolkit
- **Backend:** Flask (Python)
- **Database:** PostgreSQL
- **Design:** Figma (mobile-first wireframes)
- **Testing:** Jest & Minitests

## ✨ Features

### 👤 User Features
- Register & Login
- Browse products & services
- Add to cart & checkout
- Book veterinary services
- Track orders
- Rate & review products/services

### 🛠️ Admin Features
- Manage products (CRUD)
- Manage services (CRUD)
- Approve/reject orders
- Inventory management
- User role management
- Analytics dashboard
